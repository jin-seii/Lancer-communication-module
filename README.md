# Lancer-communication-module
This FoundryVTT module allows you to display messages on the screen, in the style of windows from various games, such as Undertale/Deltarune. The module is independent and works on any system.

You can customize many parameters for the message:

<img width="400" height="911" alt="Снимок экрана 2026-05-26 095507" src="https://github.com/user-attachments/assets/a4f9654a-d4e9-47f0-b025-30c2549ed23d" />

- Character name
- Character portrait
- Message
- Select sound for text typing (Undertale/Deltarune style)
- Select voiceover (if you want to insert a recorded sound, it overrides the sound option, even if it's selected)
- Adjust the typing speed, with an auto-tune button if you've selected a voiceover file.
- Select font
- Font size (works differently for each font)
- Message box style
(In addition to this, you can create a macro if you want to play the message later, or even create a macro for a character so you don't have to fill in other parameters every time for the same character.)

What is in the settings of the module itself?

<img width="962" height="756" alt="settings" src="https://github.com/user-attachments/assets/94c812f1-2f25-4812-aea0-8e8b7303b488" />

- You can duplicate communicator messages in chat
- You can allow a player to use a module (experimental)
- Global typing speed (set for all dialogs by default)
- Typing and voiceover volume
- Font size (will be removed)
- Text shaking when typing in all caps (may cause false positives for words that begin with a capital letter)
- Debug mode (for developers)
